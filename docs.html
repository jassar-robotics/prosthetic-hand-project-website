How to contribute

We keep docs in a public repo so anyone can improve them. If you see a gap, you can fix it quickly.

  1. Read the repo README and open issues to find what needs work.
  2. Fork the repo, make edits, and open a pull request.
  3. Add photos, settings, or test notes whenever possible.

We review contributions publicly and credit them on the site.
